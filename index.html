Work in Production

Here's a sampling of projects that I've built or currently contribute to. Due to client restrictions, codebases cannot be shared.

Here's a few examples:

Drinkbolt 24

A .net/mvc site built with a custom frontend using jQuery and the Greensock Tweenlite frameworks and packaged via grunt/node.js.

Gatorade

A site that was inherited from another agency and have steadily made updates for all of 2019 and through current. The site is built in the react framework with a seperate .net core based API for any persistence requirements.

Hello Goodness

A Sitefinity MVC project with a few custom modules and a custom theme. The theme was a jQuery and Greensock based frontend and packaged via grunt/node.js.

HapQz for Business

HapQz is a turn key solution for businesses wanting to create an online presentation. My involvement for this project was building the runtime, which can store data locally for offline usage and uses SignalR for notifications on new updates. The best example of my work is in the 'HapQz in Action' section of the homepage.

Maker Oats

An example .net/mvc site without the use of any persistence. The frontend was built in jQuery and packaged via grunt/node.js. I also have designed the hosting environment, which consists of an EC2 instance within a VPC with a public subnet.

PepsiCo Food for Good

Another .net/MVC site. This one uses a MSSQL database, entity framework for persistence. The frontend uses jQuery, Tweenlite and Grunt for code bundling.

Work in Github

I'm currently working on a series of projects with freely available source code. These examples are available for review at my Github account. If you're interested in how I would approach a project and would to see an example, please feel free to request one.

Enigma Frontend

I took the opportunity to learn how the enigma machine works and have applied the inner workings of the mechanism to a simplified backend. The frontend was built out of the React Boilerplate template project, while the meat and potatoes of the project were built as react component. The site was bundled via webpack and deployed to an S3 bucket, making the project truly serverless.

Frontend Project Github Repository

Enigma API

The backend of the Enigma Front end was developed in a c# core Web API. This was conceived as a project that can accept a simple input, apply business logic to provide an output. The scoreboard is saved to an AWS DynamoDB table. The API is deployed as a docker container and hosted via AWS Elastic Cloud Computer.
